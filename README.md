# course-vue3-vite
Vue3+vite+setup源代码

本项目是基于Vue3+vite+setup创建的项目，有引入Vuex、router、elemet-plus、axios等必备内容，可以作为Vue项目的初始结构。该项目的完整介绍可以微信搜索 "前端发现" 公众号查看。

Run 开发

    # 克隆项目
    git clone https://github.com/hzequn/course-vue3-vite.git

    # 安装依赖
    npm i
    or
    yarn


    # 本地开发 开启服务
    npm run dev
    or
    yarn dev


浏览器访问 http://localhost:3000

Build 打包

    npm run build 
    or
    yarn build
